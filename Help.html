Main Search Tab / לשונית חיפוש

The Search tab is the main entry point for exploring the Genizah corpus.

  • Query: Enter words, shelfmarks, or titles.
  • Mode: Choose how the text is matched:
    • Exact – exact word matches.
    • Variants – Hebrew-aware OCR variants.
    • Extended / Maximum – broader variant substitutions.
    • Fuzzy – Levenshtein-distance matching.
    • Regex – advanced pattern matching.
    • Title – search composition titles.
    • Shelfmark – search shelfmarks.
  • Gap: Maximum distance between query words.
  • Results Table: Displays System ID, Shelfmark, Title, snippet, and source.
  • Double-click a row to open the Manuscript Viewer.

לשונית חיפוש היא נקודת הכניסה הראשית לעבודה עם קורפוס הגניזה.

  • שאילתא: הזנת מילים, מספרי מדף או כותרות.
  • מצב: בחירת אופן החיפוש:
    • מדויק – התאמה מדויקת של מילים.
    • וריאנטים – התאמות מותאמות לשגיאות OCR בעברית.
    • מורחב / מקסימלי – חיפוש רחב יותר עם חילופי תווים.
    • עמום – חיפוש לפי מרחק לווינשטיין.
    • ביטוי רגולרי – חיפוש מתקדם באמצעות תבניות.
    • כותרת – חיפוש בכותרות חיבורים.
    • מספר מדף – חיפוש לפי מספרי מדף.
  • מרחק: המרחק המרבי בין מילות החיפוש.
  • טבלת תוצאות: מציגה מספר מערכת, מספר מדף, כותרת וקטע טקסט.
  • לחיצה כפולה על שורה פותחת את צפיין כתב היד.

Composition Search / חיפוש חיבורים

The Composition Search tool finds parallels between a source text and the Genizah corpus.

  • Paste or load a source text.
  • Chunk: Number of words per search block.
  • Max Freq: Filters out very common phrases.
  • Filter & Exclude: Remove known manuscripts or texts.
  • Results are grouped into Main results and Appendix.
  • Export a full report using Save Report.

לשונית חיפוש חיבורים מיועדת לאיתור מקבילות בין טקסט מקור לבין קורפוס הגניזה.

  • הדבקת טקסט מקור או טעינתו מקובץ.
  • גוש: מספר המילים בכל יחידת חיפוש.
  • תדירות מקס': סינון ביטויים נפוצים מאוד.
  • סינון והחרגה: הסרת כתבי יד או טקסטים ידועים.
  • התוצאות מקובצות לראשי ולנספח.
  • ניתן לייצא דוח מלא באמצעות שמור דוח.

Browse Manuscript / עיון בכתב יד

The Browse Manuscript tab is used for reading full manuscripts.

  • Enter a System ID to load a manuscript.
  • View page images and transcribed text.
  • View All shows the manuscript as one continuous text.
  • Save exports the full manuscript to a text file.

לשונית עיון בכתב יד מאפשרת קריאה רציפה של כתבי יד.

  • הזנת מספר מערכת לטעינת כתב יד.
  • צפייה בתמונות ובטקסט המתועתק.
  • הכל מציג את כתב היד כולו ברצף.
  • שמור מייצא את כתב היד לקובץ טקסט.

Settings & About / הגדרות ואודות

  • Build / Rebuild Index: Creates the search index (required on first run).
  • AI Configuration: Configure optional AI assistance.
  • About: Version, credits, and citation information.
  • בנה / בנייה מחדש של אינדקס: יצירת האינדקס (נדרש בהפעלה הראשונה).
  • הגדרות AI: הגדרת סיוע בינה מלאכותית (אופציונלי).
  • אודות: גרסה, קרדיטים ומידע לציטוט.
