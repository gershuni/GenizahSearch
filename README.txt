Genizah Search Pro 3.2.1
(עברית למטה)

1. Extract all files from the ZIP into one folder.
   Do NOT run the program from inside the ZIP.

2. Place Transcriptions.txt in the same folder as GenizahSearchPro.exe. Download it from zenodo.org/records/17734473.

3. Run GenizahSearchPro.exe.

4. On first run (and after major updates):
   Open "Settings & About" → click "Build / Rebuild Index".

See Help.html for full instructions.

---

1. חלצו את כל הקבצים מה־ZIP לתיקייה אחת.
   אין להפעיל את התוכנה מתוך ה־ZIP.

2. ודאו ש־Transcriptions.txt נמצא באותה תיקייה כמו GenizahSearchPro.exe. להורדת קובץ התעתיקים בקרו בכתובת: zenodo.org/records/17734473

3. הפעילו את GenizahSearchPro.exe.

4. בהפעלה הראשונה (וגם אחרי עדכון מגרסה 2 ל-3):
   "הגדרות ואודות" → "בנה / בנייה מחדש של אינדקס".
