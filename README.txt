Genizah Search Pro 3.5
(עברית למטה)

1. Extract all files from the ZIP into one folder.
   Do NOT run the program from inside the ZIP.

2. Download 'Transcriptions.txt' from zenodo.org/records/17734473.
   Place it next to the app OR select it manually when prompted.

3. Run GenizahSearchPro.exe.

4. On first run:
   Open "Settings & About" → click "Build / Rebuild Index".

--- What's New in v3.5 ---
* Recursive Composition Search: Run new searches based on results.
* Workflow Buttons: Jump from a result directly to "Browse" or "Parallels".
* Optimized Index: Moved to AppData (supports legacy & portable modes).
* Flexible Data Location: Locate Transcriptions.txt manually if not in the app folder.
* Browse by FL ID: Locate specific images easily.
* Smarter Exclusions: Bidirectional lookup for IDs and Shelfmarks.

1. חלצו את כל הקבצים מה־ZIP לתיקייה אחת.
   אין להפעיל את התוכנה מתוך ה־ZIP.

2. הורידו את Transcriptions.txt מהכתובת: zenodo.org/records/17734473
   שימו אותו ליד התוכנה או בחרו אותו ידנית כשהתוכנה תבקש.

3. הפעילו את GenizahSearchPro.exe.

4. בהפעלה הראשונה:
   גשו ללשונית "הגדרות ואודות" → ולחצו על "בנה / בנייה מחדש של אינדקס".

--- חדש בגרסה 3.5 ---
* חיפוש חיבורים רקורסיבי: הרצת חיפוש עומק על בסיס התוצאות שנמצאו.
* כפתורי מעבר: מעבר מהיר מצפייה בתוצאה ל"עיון בכתב יד" או ל"חפש מקבילות".
* מיקום אינדקס משופר: עבר ל-AppData (עם תמיכה במצב נייד).
* מיקום קובץ גמיש: אם קובץ התעתיקים לא נמצא, התוכנה תבקש להצביע על מיקומו.
* חיפוש לפי FL: איתור עמודים לפי מספר קובץ תמונה.
* ניהול החרגות משופר: זיהוי דו-כיווני של מספרי מערכת ומדף.
